# Jina + Streamlit

Supports both text and image search

## 1. Run Docker Image

For example:

```bash
docker run -p 45678:45678 jinaai/hub.app.distilbert-southpark
```

## 2. Install requirements

```bash
pip install -r requirements.txt
```

## 3. Start up the front end

```bash
streamlit run app.py
```
